Escopo: Ferramenta inteligente de Gestão de Talentos e Análise Preditiva de RH.
Dados Usados: Bases de dados de funcionários contendo registros biográficos e profissionais.
Resultados: 
•	Redução no tempo de consulta de informações individuais.
•	Automação do cálculo de métricas de rotatividade e tempo de casa.
•	Interface conversacional (IA) que democratiza o acesso aos dados para gestores que não dominam SQL ou Excel.
Automação de Endomarketing: Redução de 100% do esforço manual na identificação e parabenização de colaboradores aniversariantes, eliminando falhas humanas e fortalecendo a cultura organizacional.
